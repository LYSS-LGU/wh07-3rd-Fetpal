===============================================
디자인 가이드
===============================================

1. 레이아웃
   - 5개 레이어를 깔끔하게 배치
   - 화살표로 데이터 흐름 표시
   - 간격을 충분히 확보

2. 색상
   - 파스텔톤 배경색 사용
   - 테두리는 진한 색상으로 강조
   - 각 레이어마다 다른 색상

3. 폰트
   - 제목: 굵은 폰트 (Bold)
   - 내용: 보통 폰트 (Regular)
   - 크기: 제목 20pt, 내용 14pt

4. 아이콘/이모지
   - 각 항목 앞에 이모지 사용
   - 기술 로고 추가 가능 (선택사항)

5. 화살표
   - 실선: 주요 연결
   - 점선: 데이터 공유
   - 화살표 두께: 중간
   - 색상: 회색 또는 검정색

6. 여백
   - 레이어 내부 여백: 충분히
   - 텍스트 줄 간격: 1.5배
   - 레이어 간 간격: 넓게

===============================================
참고 정보
===============================================

프로젝트 개요:
- Fetpal은 AI 기반 반려동물 건강 진단 및 케어 지원 서비스입니다
- YOLO 모델로 피부/안구/건강 질환을 탐지합니다
- RAG 시스템으로 정확한 대처 방안을 제공합니다
- 실시간 채팅으로 반려인 커뮤니티를 지원합니다

주요 특징:
- Hook Composition 패턴으로 코드 60% 감소
- Supabase BaaS로 개발 속도 80% 단축
- 47개 RLS 정책으로 강력한 보안
- 4단계 반응형 디자인으로 모든 디바이스 지원
